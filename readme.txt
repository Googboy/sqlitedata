今天主要完成的是关系型轻量级数据库sqlite的操作方面：增加数据，展现数据以及删除数据的功能。展示数据有两种，一种是在控制台输出添加的数据，这种不易观察，后来
添加了一个Listview用于便于观察，添加数据手动添加，再输入框输入数据，点击添加就可以将数据添加完成，删除操作是长按文本就会弹出来删除对话框，为了防止误删除，
还添加了提醒语句。
其中遇到的问题主要有：1.使用安卓里面的simpleCursorAdapter时 必须要求数据库里面有_id这一列数据，否则出错。
2.对数据库操作完毕之后就要关闭数据库，否则会抛出SQLiteException异常。关闭数据库只需要使用.close方法就可以。
